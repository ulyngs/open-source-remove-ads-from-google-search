This Safari extension removes ads from the search results on Google.com. It is open source and collects zero data - you can find the source code on GitHub.

To turn it on, open Safari and go to google.com (or google.de, google.es, google.dk, etc). Then:
1) In the toolbar tap 'Aa' -> 'Manage Extensions', and toggle the extension on.
2) Tap the extension and allow it to access google.com
